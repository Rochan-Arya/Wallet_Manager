# Wallet Manager App

A Flutter + Firebase based wallet management application with separate User and Admin panels.

## Features

- User Login & Signup
- Wallet Dashboard
- Credit / Debit Requests
- Transaction History
- Admin Dashboard
- Approve / Reject Requests
- Edit User Wallet Balance
- Search Users
- Dark / Light Theme

## Tech Stack

- Flutter
- Firebase Authentication
- Cloud Firestore
- Provider

## Run Project

```bash
flutter pub get
flutter run
```
